==============================
  InstaLookup - Como usar
==============================

REQUISITOS:
  - Node.js 18 ou superior
  - curl instalado no sistema (já vem no Linux/Mac; no Windows use WSL)

PARA RODAR:
  1. Descompacte o ZIP em qualquer pasta
  2. Abra o terminal nessa pasta
  3. Execute:
       node server.js

  4. Acesse no navegador:
       http://localhost:3000

MUDAR A PORTA:
  PORT=8080 node server.js   (Linux/Mac)
  set PORT=8080 && node server.js   (Windows)

PRONTO! Sem npm install, sem dependências externas.
